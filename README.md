# 2048 Game

A classic 2048 puzzle game built with Taro, supporting both WeChat Mini Program and H5 platforms.

## Features

- 🎮 Classic 2048 gameplay
- 📱 Supports both WeChat Mini Program and H5
- 💾 Persistent best score storage
- 🎨 Beautiful UI with smooth animations
- ⌨️ Keyboard support for desktop (arrow keys)
- 👆 Touch swipe support for mobile

## How to Play

1. Use arrow keys (desktop) or swipe (mobile) to move tiles
2. When two tiles with the same number touch, they merge into one
3. Add up the numbers to reach 2048!

## Getting Started

### Installation

```bash
npm install
```

### Development

**WeChat Mini Program:**
```bash
npm run dev:weapp
```

**H5:**
```bash
npm run dev:h5
```

### Build

**WeChat Mini Program:**
```bash
npm run build:weapp
```

**H5:**
```bash
npm run build:h5
```
